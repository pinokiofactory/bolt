Free LLM APIs via Github

GitHub provides FREE API access to gpt-4o and gpt-4o-mini (Rate limited). All you need to do is set up the API key.

Step 1. Get Github API Key

Get API Key from GitHub

Step 2. Connect API Key

Go to the Configure tab and enter the API key, and save

Step 3. Restart Bolt

Restart Bolt and select "OpenAILike". You should see gpt4-o

LLM APIs

Directly get API keys from each LLM API provider.

Step 1. Get API Key

Go get an API key from any LLM provider. Supported LLMs
  1. Groq
  2. OpenAI
  3. Anthropic
  4. Openrouter
  5. Google
  6. Deepseek
  7. Mistral

Step 2. Connect API Key

Go to the Configure tab and enter the API key, and save

Step 3. Restart Bolt

Restart Bolt and select the LLM provider and get started!
